Troubleshooting Guide

If the extension isn't intercepting "Buy Now" buttons:

  1. Make sure the extension is enabled (toggle above)
  2. Verify you're on a supported site (Amazon, Flipkart, eBay, Walmart, Etsy)
  3. Try the "Force Inject" button above while on a shopping site
  4. Use "Inspect Active Tab" to analyze buy buttons on the current page
  5. Try hard-refreshing the page (Ctrl+F5 or ‚åò+Shift+R)
  6. Look at the debug logs below for any errors or warnings

Advanced troubleshooting:

  1. Open Chrome DevTools (F12), go to the "Console" tab, and look for ImpulseLock logs
  2. Check the Extensions page (chrome://extensions) and verify ImpulseLock is enabled
  3. Make sure you've granted all required permissions to the extension
  4. Try reinstalling the extension if all else fails
