Troubleshooting Guide

If the extension isn't working on a shopping site:

  1. Make sure the extension is enabled (toggle above)
  2. Check if the site is in our supported marketplaces list
  3. Try refreshing the page
  4. Look at the debug logs below for any errors
  5. The content script might need updated selectors for the specific website
